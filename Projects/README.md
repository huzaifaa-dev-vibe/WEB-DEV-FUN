## AI Instructions

When building a project from this folder:
1. Read the spec completely.
2. Write PLAN.md per [AI_AGENT_GUIDE.md](../AI_AGENT_GUIDE.md).
3. Pick a [Template](../Templates/).
4. Build incrementally. Verify after each step.
5. Run all applicable [Checklists](../Checklists/) before shipping.

## How to use

1. Pick a project that fits your level.
2. Read the spec.
3. Optionally, paste the AI prompt to an AI agent to scaffold.
4. Build incrementally, referencing relevant folders in this repo.
5. Run all applicable [Checklists](../Checklists/) before shipping.
