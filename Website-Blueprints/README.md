# Website Blueprints

> Per-vertical templates: structure, sections, content, design, AI prompts.

---

## Purpose

Provide ready-to-use blueprints for common website types. Each blueprint tells you: what sections, what content, what design, what AI prompt to use.

## Prerequisites

- [ANTI-AI-DESIGN/](../ANTI-AI-DESIGN/), [UI/](../UI/).

## Learning Outcome

You can ship a website for any vertical without starting from scratch.

## Dependencies

- A frontend framework.

## Related Files

- [ANTI-AI-DESIGN/](../ANTI-AI-DESIGN/) · [Projects/](../Projects/) · [Templates/](../Templates/) · [Color-Palettes/](../Color-Palettes/)

## AI Instructions

When building from a blueprint:
1. Read the blueprint completely.
2. Read [ANTI-AI-DESIGN/](../ANTI-AI-DESIGN/) before writing UI.
3. Pick a [Color-Palette](../Color-Palettes/) that fits the vertical.
4. Pick a [Template](../Templates/) for the stack.
5. Use the AI prompt at the end of each blueprint.

## How to use

1. Pick the blueprint that matches your project.
2. Read it.
3. Pick palette + template.
4. Use the AI prompt to scaffold.
5. Customize per [ANTI-AI-DESIGN/](../ANTI-AI-DESIGN/) principles.
6. Run [Checklists](../Checklists/) before shipping.
