# Minimal Palettes

> Black, white, and one accent. Swiss design.

---

## 1. Pure B&W + Red
```css
:root {
  --color-bg: #ffffff;
  --color-fg: #000000;
  --color-muted: #525252;
  --color-accent: #dc2626;
  --color-surface: #fafafa;
  --color-border: #e5e5e5;
}
.dark {
  --color-bg: #000000;
  --color-fg: #ffffff;
  --color-muted: #a3a3a3;
  --color-accent: #ef4444;
  --color-surface: #0a0a0a;
  --color-border: #262626;
}
```
