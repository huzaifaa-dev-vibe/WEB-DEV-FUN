# Professional Palettes

> Trustworthy, restrained, enterprise-friendly.

---

## 1. Editorial Slate
```css
:root {
  --color-bg: #fafaf9;       /* warm white */
  --color-fg: #1c1917;       /* warm black */
  --color-muted: #78716c;    /* warm gray */
  --color-accent: #9f1239;   /* deep rose */
  --color-surface: #ffffff;
  --color-border: #e7e5e4;
}

.dark {
  --color-bg: #0a0a0a;
  --color-fg: #fafafa;
  --color-muted: #a8a29e;
  --color-surface: #171717;
  --color-border: #262626;
  --color-accent: #f43f5e;
}
```
**Vibe:** Editorial, serious, magazine-like.

## How to extend

- Add `--color-accent-soft` (10% opacity of accent) for backgrounds.
- Add `--color-success`, `--color-warning`, `--color-error` per palette.
- Adjust contrast if needed.
