# Luxury / Niche Palettes

> Premium, editorial, brand-specific.

---

## Luxury (Champagne + Black)
```css
:root {
  --color-bg: #0a0a0a;
  --color-fg: #f5e6c8;     /* champagne */
  --color-muted: #8a7a5a;
  --color-accent: #c9a96a;  /* gold */
  --color-surface: #1a1a1a;
  --color-border: #2a2a2a;
}
```
**Vibe:** Premium, fashion, hotel.

## AI (Tech-forward, not the AI tell)
```css
:root {
  --color-bg: #0a0a0a;
  --color-fg: #fafafa;
  --color-muted: #737373;
  --color-accent: #06b6d4;   /* cyan */
  --color-surface: #171717;
  --color-border: #262626;
}
/* Optional secondary: #a3e635 (lime) for highlights */
```

> Note: AI products don't have to use the purple-pink gradient. Solid cyan/lime on dark feels more "intentional AI" than "default AI".
