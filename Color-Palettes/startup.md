# Startup Palettes

> Energetic but not childish. Modern. Friendly.

---

## 1. Indigo Pop
```css
:root {
  --color-bg: #ffffff;
  --color-fg: #0f172a;
  --color-muted: #64748b;
  --color-accent: #4f46e5;     /* indigo */
  --color-surface: #f8fafc;
  --color-border: #e2e8f0;
}

.dark {
  --color-bg: #0a0a0a;
  --color-fg: #fafafa;
  --color-muted: #a3a3a3;
  --color-surface: #171717;
  --color-border: #262626;
  --color-accent: #818cf8;
}
```
**Vibe:** Modern SaaS, friendly tech.

## 7. Royal Violet (used carefully)
```css
:root {
  --color-bg: #fafaf9;
  --color-fg: #1c1917;
  --color-muted: #78716c;
  --color-accent: #6d28d9;     /* deep violet, not pink-purple gradient */
  --color-surface: #ffffff;
  --color-border: #e7e5e4;
}

.dark {
  --color-bg: #0a0a0a;
  --color-fg: #fafafa;
  --color-muted: #a3a3a3;
  --color-surface: #171717;
  --color-border: #262626;
  --color-accent: #a78bfa;
}
```
**Vibe:** Creative, premium, intentional purple.

> Note: purple as solid color is fine. The AI tell is the **purple-to-pink gradient**.
